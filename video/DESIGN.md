## Style Prompt
AWS cloud infrastructure meets scientific instrument interface. The aesthetic bridges enterprise-grade cloud services (AWS/Vercel) with laboratory FTIR analysis — clean, precise, trustworthy. Deep navy backgrounds with teal accent convey both technical depth and chemical science. Not flashy — authoritative. The design says "this is production B2B SaaS, not a weekend prototype."

## Colors
- Background: #060E18 (deep space navy, darker than science-only palette)
- Primary Accent: #0FA5B3 (teal/cyan — chemical + AWS teal hybrid)
- Secondary Accent: #F59E0B (instrument amber — for badges, tools, highlights)
- Tertiary: #22D3EE (bright cyan — code blocks, links, active states)
- Text Primary: #E2E8F0 (warm off-white)
- Text Muted: #94A3B8 (slate, secondary text)
- Surface: #162032 (card backgrounds, elevated panels — slightly blue-tinted)
- Surface Border: #1E3A4F (subtle cyan-tinged borders)
- Success: #22C55E (DynamoDB connected, Vercel ready)
- Alert: #EF4444 (attention points)

## Typography
- Headlines: "Inter", sans-serif (700-800 weight)
- Body: "Inter", sans-serif (400 weight)
- Code/Data: "JetBrains Mono", monospace

## Motion
- Energy: Medium (corporate/SaaS explainer)
- Primary transition: Blur crossfade (0.6s, power2.inOut)
- Closing transition: Color dip to black (0.7s)
- Entrance easing: power3.out (crisp arrival)
- Stagger between elements: 0.12-0.18s

## What NOT to Do
- No neon glow or excessive bloom
- No gradient text (background-clip: text)
- No pure #000 or #fff — always tinted toward palette
- No decoration-for-decoration
- No same entrance pattern repeated within a scene
- No AWS orange as primary (too aggressive for 3 minutes) — teal primary, orange as accent only
